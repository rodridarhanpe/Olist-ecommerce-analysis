# Olist-ecommerce-analysis
Data analysis of Brazilian e-commerce dataset
Sobre el proyecto

En este proyecto analizo un dataset de e-commerce de Brasil para entender cómo se comportan las ventas, los productos y los clientes.

La idea fue practicar análisis de datos con Python y, al mismo tiempo, obtener insights que podrían servir en un negocio real.

## OBEJTIVO
- Saber cuánto vende el negocio
- Ver cómo evolucionan las ventas en el tiempo
- Analizar si el negocio crece o no
- Identificar qué productos/categorías venden más
- Entender el comportamiento de los clientes

## HERRAMIENTAS UTILIZADAS
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## PRINCIPALES RESULTADOS
## VENTAS
El negocio muestra un crecimiento general en el tiempo, aunque con algunas subidas y bajadas mensuales.

## CATEGORIAS
La categoría beleza_saude es la que más vende, lo cual tiene sentido porque son productos de consumo frecuente.

## CLIENTES
Solo alrededor del 3% de los clientes vuelve a comprar, lo que indica baja recurrencia.

Esto significa que el negocio depende mucho de conseguir nuevos clientes.

## TICKET PROMEDIO
El gasto promedio por pedido es de aproximadamente 160.

La mayoría de los pedidos son de bajo valor, pero hay algunos casos con montos muy altos.

## PEDIDOS
La cantidad de pedidos crece junto con las ventas, lo que indica que el crecimiento no depende solo del precio sino también del volumen.

## CONCLUSIONES ##

El negocio está creciendo, pero no de forma totalmente estable
Hay una fuerte dependencia de nuevos clientes
Hay oportunidades para mejorar la fidelización
Algunas categorías tienen mucho potencial

## DATASET
Brazilian E-Commerce Public Dataset by Olist
